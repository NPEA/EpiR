Abrir dados
A janela "Abrir dados" deve ser utilizada para carregar no Epi-R, bancos de dados criados tanto no R como em diversos outros softwares.

O caminho para essa janela é: Arquivo  Abrir dados. Outra forma de acessar essa função é clicar no ícone de mesmo nome na barra de atalhos da janela principal.

Figura 1. Janela "Importar arquivo de dados".

O primeiro passo é a localização do arquivo. Ao clicar em "Localizar", a próxima janela será aberta:


Figura 2. Janela auxiliar para localizar bancos de dados.


É necessário então, indicar o caminho onde o banco de dados está armazenado, como mostra a Figura 2.

Informações sobre os botões "Adicionar" e "Remover", clique aqui.

O Epi-R permite que sejam abertos bancos de dados criados por diversos softwares. A lista com essas opções aparece no campo localizado no canto inferior direito da Figura 2.

Uma vez que o banco tenha sido selecionado, clique no botão "Abrir". Com isso, essa janela auxiliar será fechada, retornando à janela da figura 1.

Observação: O Epi-R não lê arquivos com extensão *.xls (arquivos originais do Microsoft Excel). A opção mais simples para ler um arquivo desse tipo é salvá-lo no formato *.csv (arquivo separado por vírgula). Os arquivos com essa extensão são facilmente lidos no Epi-R.

Para verificar se o banco de dados está sendo importado corretamente, clique em "Visualizar". Se estiver faltando alguma informação, ou o Epi-R não estiver conseguindo identificar o formato correto para o banco, uma mensagem de aviso será apresentada nessa janela.

Depois de informar o banco, o passo seguinte é atribuir um nome ao objeto, que não precisa ser igual ao nome do banco selecionado. Isso é feito preenchendo o campo "Salvar como". Com isso, ao abrir o objeto, esse nome aparecerá no campo "Objetos" da janela principal.

Para alguns formatos de arquivo, é necessário especificar o caractér que indica a separação de variáveis e o caractér que indica a separação decimal. Quando arquivos desses tipos forem selecionados, esses dois campos serão ativados na janela da figura 1. Depois de preencher essas informações, basta clicar em "Executar".

Para os outros tipos de arquivo, após selecioná-lo, basta clicar em "Executar" que a janela será fechada e o objeto aparecerá disponível no campo "Objetos" da janela principal.
