Importar dados via ODBC
A janela "Importar dados via ODBC" permite acessar bancos de dados no Epi-R, utilizando uma conexão ODBC (Open Database Connectivity). Para isso, é necessário que o computador já possua uma conexão desse tipo configurada. Para maiores informações, contate o administrador do seu sistema.

O caminho para acessar essa função é: Arquivo  Importar dados via ODBC.

Figura 1. Janela "Importar dados via ODBC".

Para realizar a conexão, o primeiro passo consiste em preencher os seguintes dados:

Fonte de dados: É o nome da conexão ODBC do seu sistema. Para maiores informações, contate o administrador do seu sistema.

Usuário: Como o nome diz, é o login do usuário, que acessa a conexão.

Senha: É a senha do banco de dados. É importante lembrar que nem todos os bancos precisam de senhas. Assim, esse campo é opcional.

Maiúsculas/Minúsculas: Cada servidor tem uma forma de lidar com caracteres. É necessário especificar a forma utilizando esse campo. As opções são: não mudar; tudo em maiúscula; tudo em minúscula; linguagem mysql; linguagem postgresql; liguagem oracle; e liguagem msacess.

Feito isso, clique em "conectar" para realizar a conexão. Se tudo estiver correto, ao lado do "Estado da conexão" um botão verde ficará acesso no lugar do botão vermelho que pode ser visto na Figura 1.

O próximo passo é a construção da consulta em SQL. Em seguida, um nome deve ser atribuído ao objeto que será gerado pela consulta. Esse nome deve ser escrito no campo "Salvar como" e não há valor padrão.

Por fim, se desejar construir mais de uma consulta, sem que seja necessário estabelecer novamente a conexão, habilite a opção "Manter a janela aberta", e execute a operação.
