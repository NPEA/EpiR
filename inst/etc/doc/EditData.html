Visualizar dados
A janela "Visualizar dados" apresenta um objeto que é do tipo banco de dados no formato tabular. Permite também que esse objeto seja editado manualmente.

O caminho para essa janela é: Dados Visualizar dados.

Figura 1. Janela "Visualizar dados".

Para visualizar um banco de dados, o primeiro passo é indicar o objeto desejado. Para isso, é necessário que o banco já tenha sido aberto (detalhes sobre como abrir dados ou importar dados via ODBC).

Ao selecionar um objeto de dados, automaticamente as 100 (cem) primeiras linhas e colunas desse objeto serão mostradas. As demais linhas poderão ser exibidas clicando-se no botão "Próximo" ou "Anterior" que mostrarão as cem próximas observações ou as cem anteriores, respectivamente. As últimas cem observações podem ser visualizadas clicando-se no botão "Última" e, caso o interesse seja visualizar as cem primeiras observações do banco de dados, basta clicar no botão "Primeira".

Para editar o objeto, um nome deve ser atribuído a esse novo banco, utilizando o campo "Salvar como". O nome sugerido pelo Epi-R é o mesmo nome do objeto. Deve-se lembrar que ao utilizar o mesmo nome, o objeto será substituído. Com isso, se alguma casela for modificada de forma errada, a única maneira de recuperar as informações anteriores será abrindo novamento o objeto de origem. Para editar o objeto selecionado, clique em "Editar banco de dados". Outra tela será aberta e as alterações deverão ser feitas nela.

Quando as alterações estiverem concluídas, feche o editor e clique em "Executar" para salvá-las. Para cancelar as alterações e manter o banco como o de origem, clique em "Cancelar".
