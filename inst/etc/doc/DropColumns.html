Excluir colunas
A opção de excluir colunas, associada a opção de seleção de linhas, permite a criação de filtros para objetos de dados que tenham sido abertos no Epi-R.

Para acessar essa função, clique em Dados Exluir colunas.

Figura 1. Janela "Excluir colunas".

Para excluir algumas variáveis, a janela utilizada é apresentada na Figura 1. A primeira informação que deve ser indicada se refere ao objeto de dados que será utilizado. Deve-se lembrar que para realizar esse filtro, o objeto de dados deverá ter sido previamente aberto no Epi-R (detalhes sobre como abrir dados ou importar dados via ODBC).

Em seguida, deverá ser informado o nome que será atribuído ao objeto de dados resultante da operação. O nome sugerido pelo Epi-R é o mesmo do objeto de origem dos dados. Lembre-se que ao utilizar o mesmo nome, o objeto será substituído. Com isso, as colunas excluídas só poderão ser recuperadas se o objeto for aberto novamente.

Depois de selecionar o objeto de dados, todas as suas variáveis aparecerão no campo "Selecione as variáveis". Para marcar uma variável, clique sobre ela. Se desejar selecionar mais de uma variável, pressione "Ctrl" e clique sobre as variáveis desejadas.

Algumas vezes, o interesse pode ser em manter apenas poucas variáveis. Nesses casos, em vez de marcar todas as variáveis que deseja excluir, existe a opção de selecionar apenas aquelas que devem permanecer no banco. Para isso, clique nas variáveis que deseja manter no objeto e habilite a caixa ao lado de  "Todas exceto as selecionadas".

Quando todas as variávies estiverem sido escolhidas, clique em "Executar".
