Salvar resultados
A janela "Salvar resultados" é utilizada para salvar em um arquivo texto (extensão *.txt) o conteúdo do campo "Saída de resultados" da janela principal do Epi-R. Para saber como abrir esses resultados, clique aqui. 

O caminho para essa janela é: Arquivo  Salvar resultados.

Figura 1. Janela "Salvar resultados".

Para salvar um arquivo de resultados, o primeiro passo é indicar o nome que se deseja atribuir ao arquivo, no campo "Nome". O nome padrão adotado pelo Epi-R é o nome do diretório raiz, seguido da data atual no formato ano, mês, dia e hora. Para alterar essa informação, basta escrever o nome desejado.

A próxima informação consiste em indicar onde o arquivo deverá ser armazenado, utilizando o campo "Salvar na pasta".

Informações sobre os botões "Adicionar" e "Remover", clique aqui.

Depois de preencher essas informações, basta clicar em "Salvar".

Como o arquivo salvo é do tipo texto, ele poderá ser aberto utilizando o Epi-R ou qualquer programa de edição de texto.
