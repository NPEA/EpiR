Salvar histórico como script
Ao concluir alguma operação, tanto escrevendo no campo "Entrada de comando" da janela principal como utilizando os menus, a função (ou conjunto de funções) utilizada é enviada para o campo "Histórico de comandos" da janela principal.

Para salvar essa seqüência de operações, uma opção é salvar como um arquivo de histórico. Esse arquivo poderá ser aberto em outras sessões do Epi-R, porém, apenas esse aplicativo utiliza arquivos com extensão "*.his".

Se desejar salvar esse arquivo de forma que ele possa ser editado em outros programas, como editores de texto ou de script (ex. Tinn-R), o usuário deve utilizar a opção: menu Arquivo Salvar histórico como script. A Figura 1 apresenta a janela que será aberta.

É importante lembrar que esse arquivo com extensão "*.r" não poderá ser recuperado pela função carregar histórico. Esse tipo de arquivo permite que o usuário armazene e/ou faça alterações nos comandos utilizados. Se desejar utilizar o conteúdo desse arquivo novamente, a opção é copiá-lo para o campo "Entrada de comando" da janela principal e em seguida clicar em "Executar".

Figura 1. Janela "Salvar histórico como script".

Ao abrir a janela, a primeira informação que o usuário deve preencher é o nome que será atribuído ao arquivo de histórico. Para isso, basta escrever o nome desejado no campo "Nome". O nome padrão adotado pelo Epi-R é o nome do diretório raiz, seguido da data atual no formato ano, mês, dia e hora. Para alterar essa informação, basta escrever o nome desejado

É importante lembrar que todas as linhas de comando que estiverem aparecendo no campo "Histórico de comandos" serão salvas nesse arquivo.

A próxima informação consiste em indicar onde o arquivo deverá ser armazenado. Selecionar o diretório no campo "Salvar na pasta".

Informações sobre os botões "Adicionar" e "Remover", clique aqui.

Depois de indicar o nome e o diretório para o arquivo, clique em "Salvar".
