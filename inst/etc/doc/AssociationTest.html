Teste de Associação
O caminho para essa função é: Análises Testes de hipóteses Teste de associação


Figura 1. Janela "Teste para Associação".

 Para a construção de um teste de hipóteses de associação, siga os seguintes passos:

1. Selecione o tipo de teste que deseja realizar. Use "Qui-Quadrado" se quiser verificar a associação entre duas variáveis categóricas, cada uma contendo pelo menos duas categorias; e "Mantel-Haenszel" se desejar verificar se duas variáveis categóricas são  independentes em cada estrato, assumindo que há uma interação entre as três.

Essa seleção habilitará algumas opções e deixará desabilitadas aquelas que não serão utilizadas.

2. Selecione o banco de dados no campo "Banco de dados". Para isso, é necessário que o banco já tenha sido aberto (detalhes sobre como abrir  abrir dados ou importar dados via ODBC).

3. Selecione as variáveis de interesse nos campos "Variável 1" e "Variável 2". Nesses campos, todas as variáveis com até dez valores diferentes serão apresentadas.

4. Para selecionar uma parte do banco, use o campo "Filtro". Esse filtro deverá seguir os mesmo padrões de filtro do R.

5. A opção "Estrato" estará habilitada apenas no teste "Mantel-Haenszel".

6. O "Nível de confiança" normalmente utilizado é de 95%.

7. Determinar as hipóteses do teste. A hipótese nula sempre será de igualdade (ou com o valor, ou entre as variáveis). É preciso então determinar a hipótese alternativa, ou seja, o conjunto de valores complementares àquele considerado na primeira hipótese. Essa opção só está habilitada para o teste de Mantel-Haenszel

8. Se desejar observar a Tabela de Contigência, basta marcar a opção "Mostrar tabela".

9. A opção "Simular valor-p" utiliza a técnica de simulação de Monte Carlo

Para realizar sucessivos testes usando essa janela, marque a opção "Manter essa janela aberta para outra operação". Com isso, o resultado desejado aparecerá no campo "Saída de resultados" da janela principal do Epi-R, mas essa janela não será fechada, evitando o retrabalho de acessá-la.
