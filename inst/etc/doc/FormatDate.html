Formatar datas

O caminho para essa função é: Dados Formatar datas.


Figura 1. Janela "Formatar datas".

Essa janela deverá ser usada para transformar uma variável que representa uma data, mas que não está no formato compreendido pelo R. Para construir uma nova variável de data, siga os seguintes passos:

1. Selecione o banco de dados onde está a variável de interesse no campo "Banco de dados". Para isso, é necessário que o banco já tenha sido aberto (detalhes sobre como abrir dados ou importar dados via ODBC).

2. Selecione a variável que representa a data no banco de dados original e atribua um nome para a nova variável que será criada nos campos seguintes.

3. Ao selecionar uma variável, no campo "Visualização", dentro de "Formato original", aparecerá a pirmeira informação válida. A partir dessa visualização, preencha os campos referentes à forma de apresentação dos dias, meses, anos e separadores. Isso será usado para recodificar a data. Para conferir se a nova data está sendo criada corretamente, clique no botão "Visualizar".

Depois de confirmar a visualização da nova variável, clique em "Executar". Para criar mais de uma variável usando essa janela, marque a opção "Manter essa janela aberta para outra operação". Com isso, a variável será criada, mas essa janela não será fechada, evitando o retrabalho de acessá-la.
