Selecionar linhas
A opção de selecionar linhas, associada a opção de excluir colunas, permite a criação de filtros para objetos de dados que tenham sido abertos no Epi-R.

Para acessar essa função, clique em Dados Selecionar linhas.

Figura 1. Janela "Selecionar linhas".

Para selecionar algumas linhas, a janela utilizada é apresentada na Figura 1. A primeira informação que deve ser indicada é referente ao objeto de dados que será utilizado. Deve-se lembrar que para realizar esse filtro, o objeto de dados deverá ter sido previamente aberto no Epi-R (detalhes sobre como abrir dados ou importar dados via ODBC).

Em seguida, deverá ser informado o nome que será atribuído ao objeto de dados resultante da operação. O nome sugerido pelo Epi-R é o mesmo do objeto de origem dos dados, adicionado de "novo_" antes do nome do objeto, podendo ser manualmente alterado pelo usuário. Lembre-se, no entanto, que ao utilizar o mesmo nome do banco de dados de orgiem, o objeto será substituído. Com isso, as linhas excluídas só poderão ser recuperadas se o objeto for aberto novamente.

Depois de selecionar o objeto, todas as variáveis aparecerão no campo variável do construtor de filtros. O próximo passo será construir o(s) filtro(s) desejado(s). Para isso:
1. Selecione a variável que será utilizada para a seleção das linhas;
2. No campo "Operador relacional", indique que tipo de filtro deseja utilizar. Por exemplo, para selecionar registros de idosos, o operador relacional >= ou > deverá ser utilizado para selecionar apenas pessoas com mais de 65 anos;
3. Ao selecionar uma variável, todos os valores dessa variável serão apresentados no campo "Valores". Marcar o corte que será utilizado. No exemplo apresentado acima, 65 ou 66, dependendo do operador relacional utilizado.
4. O uso dos parênteses é opcional e pode ser utilizado em conjunto com os operadore "E" e "Ou". 
5. Clique em "Adiciona" e a linha de comando será transferida para o campo "filtros definidos".

Se algum filtro estiver errado, basta clicar sobre ele no campo "Expressão" e em seguida, editá-lo manualmente. Para apagar todo o conteúdo desse campo, clique em "Limpar".

Quando a seleção desejada estiver sido criada, basta clicar em Executar.
