Bem-vindo ao menu de ajuda do Epi-R.
Aqui se encontram informações sobre todas as funções disponíveis no aplicativo. Todas as janelas apresentam um botão de que direciona a página diretamente à função de interesse. Além disso, sempre que possível, um exemplo é apresentado logo após a sua descrição.

O Epi-R está alinhado com a crescente disseminação em todo o mundo de soluções computacionais de código aberto e gratuitas (open source), oferecendo como principais vantagens o baixo custo e a livre auditoria, já que seu código está disponível para a consulta dos algoritmos utilizados.

Seu desenvolvimento está completamente baseado em softwares livres como SQL para gerenciamento de dados, R para a análise de dados, GTk/Glade para a construção de interfaces gráficas com o usuário e C++ como linguagem de programação de baixo nível. O Epi-R está sendo desenvolvido como um pacote para o R e é compatível com Windows e Linux.
