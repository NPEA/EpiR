Remover dados
Todos os objetos que são abertos ou criados durante uma sessão do Epi-R, ficam armazenados no aplicativo e são mostrados no campo "Objetos de dados" da janela principal. A função "Remover dados" permite que alguns objetos ou todos sejam deletados.

O caminho para essa janela é: Dados Remover dados.

Figura 1. Janela "Remover dados".

Para selecionar um objeto, clique sobre ele no campo "Selecione os objetos para remover" da janela apresentada na Figura 1. Se desejar selecionar mais de um, pressione "Ctrl" e clique sobre os objetos desejados.

Para selecionar todos os objetos, clique em "Selecionar todos".

Em seguida, clique em "Executar".

Uma outra forma de apagar todos os objetos sem precisar entrar nessa janela, é clicar em Dados Remover tudo.

Uma opção interessante é a possibiliade de recuperar dados apagados. Essa opção é dada pelo assistente Gerenciar lixeira no menu Dados. É importante  lembrar que ao encerrar uma sessão do Epi-R, os dados que estiverem na lixeira serão automaticamente deletados, não permitindo recuperação.
