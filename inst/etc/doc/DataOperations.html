Transformar dados
A janela "Transformar dados" permite a criação de novas variáveis dentro de objetos de dados.

O caminho para essa janela é: Dados Operações com dados.

Figura 1. Janela "Transformar dados".

Para criar uma nova variável, a janela utilizada é apresentada na Figura 1. A primeira informação que deve ser indicada se refere ao objeto de dados que será utilizado. Deve-se lembrar que para realizar esse filtro, o objeto de dados deverá ter sido previamente aberto no Epi-R (detalhes sobre como abrir dados ou importar dados via ODBC).

Em seguida, deverá ser informado o nome que será atribuído ao objeto de dados resultante da operação. O nome sugerido pelo Epi-R é o mesmo do objeto de origem dos dados. Lembre-se que ao utilizar o mesmo nome, o objeto será substituído. Assim, ao criar uma nova variável, se desejar excluí-la para recuperar o objeto de dados de origem, isse só será possível abrindo novamente o objeto ou excluindo a(s) coluna(s).

O próximo passo é construir a(s) variável(is) desejada. Isso será feito utilizando o "Construtor de operações". Para isso:

1. Indique o nome da variável que será criada;

2. Construa a expressão desejada para a nova variável. Você pode fazer isso manualmente, digitando a expressão no campo "Expressão", ou utilizando a "Calculadora". Para maiores informações sobre a calculadora, clique aqui.

3. Ao criar a expressão para uma variável, clique em "Adiciona". Com isso, é possível construir mais de uma variável ao mesmo tempo.

Se alguma variável estiver errada, basta clicar sobre ela no campo "Operações definidas" e em seguida, clicar em "Excluir".

Quando todas as variáveis estiverem sido construídas e adicionadas, clique em "Executar".
