Correlação
A função de correlação deve ser usada para calcular correlações entre variáveis duas a duas.

O caminho para essa função é: Análises  Análises descritivas  Correlação.


Figura 1. Janela "Correlação".

O primeiro passo é selecionar o banco de dados onde está a variável que se deseja descrever. Para isso, é necessário que o banco já tenha sido aberto (detalhes sobre como abrir dados ou importar dados via ODBC).

Em seguida, as variáveis numéricas aparecerão disponíveis no campo "Variáveis". Selecione pelo menos duas variáveis. Ao selecionar mais de duas, os resultados serão apresentados para todos os pares possíveis de variáveis.

Um item opcional da janela é a variável de estratificação.Quando o banco de dados é selecionado, as variáveis que podem ser utilizadas como fatores de estratificação aparecerão no campo "Estratificação". É permitida a escolha de apenas uma variável de estratificação.

O próximo passo é a determinação de qual método será utilizado para calcular a correlação. Os métodos disponíveis no Epir-R são a correlação de Pearson, de Kendall e de Spearman. Não é possível escolher mais de um método ao mesmo tempo. Para isso, utilizae o campo "Manter esta janela aberta para outra operação" que está apresentado abaixo.

Depois de selecionar todas as opções desejadas, basta clicar em "Executar". Para fazer sucessivas análises usando essa janela, marque a opção "Manter essa janela aberta para outra operação". Com isso, o resultado desejado aparecerá no campo "Saída de resultados" da janela principal do Epi-R, mas essa janela não será fechada, evitando o retrabalho de acessá-la.
