Teste para variância
O caminho para essa função é: Análises Testes de hipóteses Teste para variância


Figura 1. Janela "Teste para Variância".

 Para a construção de um teste de hipóteses para a variância, siga os seguintes passos:

1. Selecione o tipo de teste que deseja realizar. Use o teste "F" se deseja comparar a variância de duas amostras oriundas de populações com distribuição Normal.

Caso queira testar a homogeniedade das variâncias, isto é, testar se as variâncias de cada grupo (amostra) são iguais, marque a opção "Bartlett".

O teste "Fligner" tem a utilização muito parecida que o teste "Bartlett". Para maiores informações: William J. Conover & Mark E. Johnson & Myrle M. Johnson (1981). A comparative study of tests for homogeneity of variances, with applications to the outer continental shelf bidding data. Technometrics 23, 351–361.

Essa seleção habilitará algumas opções e deixará desabilitadas aquelas que não serão utilizadas.

2. Selecione o banco de dados no campo "Banco de dados". Para isso, é necessário que o banco já tenha sido aberto (detalhes sobre como abrir dados ou importar dados via ODBC).

3. Selecione as variáveis de interesse nos campos "Variável 1" e "Variável 2". Nesses campos, todas as variáveis com até dez valores diferentes serão apresentadas.

4. Para selecionar uma parte do banco, use o campo "Filtro". Esse filtro deverá seguir os mesmo padrões de filtro do R.

5. Determine o valor do "Parâmetro" que será testado (esta função está apenas habilitada para o teste "F"). O padrão do Epi-R para o teste é 1.

6. O "Nível de confiança" normalmente utilizado é de 95%.

7. Determinar as hipóteses do teste. A hipótese nula sempre será de igualdade (ou com o valor, ou entre as variáveis). É preciso então determinar a hipótese alternativa, ou seja, o conjunto de valores complementares àquele considerado na primeira hipótese.

Para realizar sucessivos testes usando essa janela, marque a opção "Manter essa janela aberta para outra operação". Com isso, o resultado desejado aparecerá no campo "Saída de resultados" da janela principal do Epi-R, mas essa janela não será fechada, evitando o retrabalho de acessá-la.
