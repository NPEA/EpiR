Combinar bancos
A função "Combinar bancos" tem o objetivo de auxiliar na conexão de dois bancos de dados. Essa operação pode ser feita tanto a partir da inclusão de novas variáveis como da adição de linhas ao banco original.

Para acessar essa função, clique em Dados Combinar bancos.

Figura 1. Janela "Combinar bancos".

Para utilizar essa função, é necessário que os bancos de dados que serão combinados já tenham sido abertos previamente no Epi-R. Para isso, utilize as funções Abrir dados ou Importar dados via ODBC.

A primeira informação que deverá ser preenchida na janela apresentada na figura 1 se refere ao banco de dados que servirá de base para a combinação. Para isso, selecione um banco de dados no campo "Banco de dados" da coluna "Dados originais". Posteriormente, repita essa ação incluindo o segundo banco de dados que será utilizado, preenchendo o mesmo campo da coluna "Dados novos".

Em seguida, é preciso determinar que tipo de combinação será realizada. As orientações seguem abaixo:

Combinar bancos a partir da inclusão de novas variáveis.
1. Selecione a variável de ligação em cada banco de dados, completando o campo referido nas colunas "Dados originais" e "Dados novos".
2. Escolha as variáveis que devem pertencer ao banco de dados de resultados, selecionando-as no campo "Seleção de variáveis", também em ambas colunas.

Combinar bancos a partir da adição de linhas (registros).
1. Marque a opção "Agrupar mesmas variáveis". Ao fazer isso, as opções da coluna "Dados novos" serão desabilitadas.
2. Escolha as variáveis que devem pertencer ao banco de dados de resultados, selecionando-as no campo "Seleção de variáveis" da coluna "Dados originais".

Por fim,

Depois de selecionar todas as opções, basta clicar em "Executar". Para fazer sucessivas combinações de dados usando essa janela, marque a opção "Manter essa janela aberta para outra operação". Com isso, o novo banco será construído e ficará disponível no campo "Objetos" da janela principal do Epi-R, mas essa janela não será fechada, evitando o retrabalho de acessá-la.
